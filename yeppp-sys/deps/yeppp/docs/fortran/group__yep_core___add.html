interface yepcore::yepCore_Add_V8sV8s_V8s

Adds corresponding elements in two signed 8-bit integer arrays. Produces an array of signed 8-bit integer elements.

Parameters
[in]xThe first addend array of signed 8-bit integer elements.
[in]yThe second addend array of signed 8-bit integer elements.
[out]sumThe summand array of signed 8-bit integer elements.
[in]lengthLength of the arrays specified by x, y, and sum.
Return values
0The computation finished successfully.
2x, y or sum argument is not naturally aligned.
